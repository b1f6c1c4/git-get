# git-download

```bash
git-download
    [https://github.com/]<user>/<repo>
    [<branch>|<sha1>]
    [-o <target> [-f|--force] [-F|--rm-rf]]
    [-r|--recursive] [--] [<path>]
```

Supports bash _and PowerShell TODO_.

## Install

```bash
git-download b1f6c1c4/git-download -o /usr/bin/git-download --force
```
